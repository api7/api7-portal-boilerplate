# Introduction

This is the **API7 Developer Portal** — a Next.js app that fronts the API7
Enterprise Portal API. Developers browse the API Hub, create applications,
manage credentials, and subscribe to API products.

It also ships a built-in **documentation system**: a Markdown (MDX) docs site at
`/docs` (these pages). It is file-driven and styled with the same design tokens
as the rest of the portal, so it looks like one product.

## In these docs

- [Running the project](/docs/getting-started) — install, configure, and start the dev server.
- [Writing documentation](/docs/writing-docs) — how to add and organize `/docs` pages.

<Callout type="info" title="Editing these pages">
  Everything you see here lives in `apps/site/content/docs/`. Edit a file and
  the page updates automatically in the dev server.
</Callout>
